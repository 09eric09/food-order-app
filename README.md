# Food order app with ReactJS

Food order App with cart functionality. Sends orders to Firebase. Made with React JS.

### Installing

Clone the Repository and run

```
npm install
npm start
```

## Built With

- [React JS](https://reactjs.org/)
- [Firebase](https://firebase.google.com/)
